# Roadmap: InsightPulse

## Product Vision

InsightPulse aims to become a lightweight AI feedback intelligence platform that helps product, engineering, and customer-facing teams turn customer feedback into faster product decisions.

---

## Phase 1: MVP Prototype

**Status:** Launched / Public Preview

Completed MVP capabilities:

- Public app access
- Feedback dashboard
- Review submission
- Mock data import
- Recent reviews feed
- Sentiment classification
- Category detection
- Urgent issue detection
- AI dashboard summary agent

Goal:

Validate the core workflow: customer feedback → AI triage → dashboard insight → product summary.

---

## Phase 2: Analytics Improvements

Planned improvements:

- Add date-based sentiment trends
- Add dashboard filters by sentiment and category
- Add category-level drilldowns
- Add urgent issue panel
- Add exportable product insight reports

Product value:

Help Product Managers quickly compare feedback patterns and identify the most important product areas.

---

## Phase 3: Workflow Integrations

Planned improvements:

- Add CSV upload
- Add support-ticket ingestion
- Add Jira, Azure DevOps, or Linear integration
- Convert urgent reviews into engineering tickets
- Add app-store or review-platform ingestion

Product value:

Move from dashboard-only insight to action-oriented product and engineering workflows.

---

## Phase 4: AI Governance

Planned improvements:

- Add AI confidence scores
- Add human review and correction flow
- Track sentiment and category accuracy
- Explain why a review was marked urgent
- Add feedback loop for improving AI labels

Product value:

Improve user trust and make AI-generated insights more transparent and reliable.

---

## Phase 5: Enterprise Readiness

Planned improvements:

- Add team workspaces
- Add role-based access control
- Add audit logs
- Add customer/account-level reporting
- Add admin settings
- Add data retention controls

Product value:

Prepare the product for larger B2B SaaS teams with security, governance, and collaboration needs.

---

## Future Success Metrics

- Reduce manual feedback review time
- Improve urgent issue detection speed
- Increase number of feedback items converted into backlog actions
- Improve product and engineering alignment
- Improve user trust in AI-generated summaries
