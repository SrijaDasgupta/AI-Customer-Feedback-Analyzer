# InsightPulse — AI Customer Feedback Analyzer

**Live App:** https://sense-feedback-flow.base44.app
**Status:** MVP Prototype / Public Preview

## Overview

InsightPulse is an AI-powered customer feedback dashboard that helps product and engineering teams turn raw customer reviews into sentiment insights, product categories, urgent issue flags, and AI-generated dashboard summaries.

This project was created as an AI-assisted B2B SaaS prototype to demonstrate Technical AI Product Management skills, including product strategy, AI workflow design, dashboard analytics, feedback triage, and roadmap prioritization.

## Key Features

* Customer review submission
* Mock data import
* Sentiment classification
* Category detection: Bug, Feature Request, UI/UX, Performance
* Urgent issue detection
* Recent reviews feed
* Product health dashboard
* AI dashboard summary agent

## Target Users

* Product Managers
* Engineering Leads
* Customer Success Teams
* Support Teams
